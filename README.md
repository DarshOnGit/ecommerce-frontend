# Ecommerce Frontend

A React + Vite ecommerce frontend with cart, checkout, order tracking, and search functionality.

## Tech Stack
- React 19
- React Router v7
- Axios
- Vite
- Day.js

## Features
- Product listing with search
- Add to cart / update quantity
- Checkout page with delivery options
- Orders page
- Order tracking with progress bar

## Setup
npm install
npm run dev
